TerminalNote — marks that an interactive terminal session was handed off.

```jsx
<TerminalNote>Terminal session opened — continue the discussion there.</TerminalNote>
<TerminalNote />  {/* defaults to "Terminal session opened." */}
```

Appears after "Discuss in terminal" / "Open in terminal" / "Resolve in terminal" actions. The `›_` glyph is the system's terminal marker — keep it.
