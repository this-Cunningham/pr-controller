Usage notes

Reach for motion whenan element arrives, leaves, or is live — feedback the user should notice. Static state needs no animation.

Ambient vs. transitionOne-shot transitions use these tokens. Long ambient loops belong to OrganicLoader, whose per-glyph timing is intrinsic and stays literal — show one at a time, never racing.

Reduced motionNothing springs or overshoots. prefers-reduced-motion collapses every animation to near-instant; each component's visible end-state is its base style, so nothing is load-bearing.
